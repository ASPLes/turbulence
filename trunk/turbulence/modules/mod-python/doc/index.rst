Pyturbulence's documentation center!
====================================

The following pages includes API documentation to use PyTurbulence API
which is implicitly included in all mod-python apps under the
tbc namespace.

Once a mod-python's application is loaded, you can use all the API
here described directly access the "tbc." namespace, which in turn
represents the TurbulenceCtx (:class:`tbc.Ctx`) that is representing the particular
execution. See :ref:`examples` for more details.
