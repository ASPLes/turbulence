Turbulence documentation

Under this section is located all turbulence documentation available. It includes general manuals to configure turbulence and some specific documents to properly deploy some turbulence modules that are included in the official distribution.

Base documentation:

  1. Installing turbulence: some notes about Turbulence installation.
  2. Configuring turbulence (general options): Turbulence configuration manual. Still not complete.

Modules documentation:

  1. mod-sasl: SASL support for Turbulence: providing user autentication to your Turbulence server.
  2. mod-tunnel: TUNNEL (RFC3620) support for Turbulence: building a BEEP proxy server.

Extending Turbulence:

  1. Creating turbulence modules: a sort tutorial explaining how looks like a module and how to built it.
  2. Vortex Library Documentation Center: this is a really important source of documentation. It is the API used from inside modules created to extend and configure Turbulence behaviour.
  3. Turbulence API: additional set of functions provided to modules and tools on top of Turbulence.
