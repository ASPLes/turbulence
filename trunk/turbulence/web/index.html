Turbulence home page

Turbulence is a general BEEP server built on top of Vortex Library, that provides many site administration features. Turbulence is extensible through modules and allows to implement server side profiles that are used and combined later with other profiles through run time configuration.

Turbulence is provided as free application released under the GPL, and it is developed and maintained by ASPL.
